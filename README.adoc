= FIPS Compliance with Container Images

This lab demonstrates how Red Hat Hardened Images enforce FIPS 140-3 cryptographic
algorithm restrictions at the container level, independent of host configuration.

== Lab Overview

Participants examine FIPS compliance requirements, check host FIPS state, then build
and compare the same Python application in standard and FIPS-enabled image variants
to observe algorithm enforcement in action.

== Modules

* Module 01: FIPS Context — understand the compliance landscape and host state
* Module 02: Compare Enforcement — build and run both variants, compare output

== Duration

12-15 minutes
